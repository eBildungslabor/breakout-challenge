# BreakOut-Challenge

Die BreakOut-Challenge ist eine austauschorientierte Methode, die in Videokonferenzen genutzt werden kann. Teilnehmende werden per Zufall in BreakOut-Gruppen eingeteilt - und müssen dort gemeinsam versuchen, mögichst viele der vorgestellten Herausforderungen zu bewältigen. Die Gruppe, die die meisten Herausforderungen bewältigt, gewinnt.
